Microframework

  • Provide a very simple glue layer to the best that Python has to offer.
  • Micro with Dependencies.
  • One Template Engine.
  • Werkzeugo-based WSGI and Automatic Routing Systemat.
  • Thread Locals.
  • Flask will never have a database layer.
  • It will not have a form library or anything else in that direction.

Explicit Application Object

from flask import Flask
app = Flask(__name__)

@app.route('/')
def index():
    return 'Hello World!'

Instead of implicit

from hypothetical_flask import route

@route('/')
def index():
    return 'hello world!'

Explicit/implicit

  • Implicit application objects require that there may only be one instance at the time (application factories).
  • Flexible package name and modularization.
  • Explicit is better than implicit.

Develop for the Web with Caution

  • Always keep security in mind when building web applications.
  • There are many ways the security of a web application can be compromised.
